# ☁️ Cloud Storage Drive

## 📌 Project Overview

Cloud Storage Drive is a web-based cloud storage application developed to provide secure and efficient file management using cloud technology. The system allows users to upload, download, delete, and manage files online through a simple web interface. The project uses AWS S3 cloud storage for storing files securely and provides user-specific folders to maintain data privacy and organization.

The project is designed to demonstrate practical implementation of cloud computing concepts such as cloud storage, user authentication, file management, and client-server architecture.

---

# 🚀 Features

- User Registration and Login
- Secure Authentication System
- File Upload and Download
- User-Specific Cloud Storage
- File Delete Functionality
- AWS S3 Cloud Integration
- Scalable Cloud-Based Storage
- Simple and User-Friendly Interface
- Backup Support

# 🌐 Deployment

The application is deployed on AWS EC2 and integrated with AWS S3 cloud storage services.


👉 “Please make the necessary changes according to your EC2 instance IP address, S3 bucket name, and AWS region. Also, attach an Elastic IP to the EC2 instance so that the public IP address does not change after restarting the server.”
